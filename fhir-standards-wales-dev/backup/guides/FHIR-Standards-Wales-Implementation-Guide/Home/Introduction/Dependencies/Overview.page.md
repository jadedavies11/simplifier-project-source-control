### {{page-title}}

All FHIR projects have a basic dependency on HL7 FHIR. Additionally, projects may have other dependencies that are contingent on specific local requirements. 

Dependencies enable synchronisation with the packages they are derived from without the need to add separate files and incur additional maintenance. 

To support a common approach to the creation and maintenance of HL7 FHIR standards across the four nations, FHIR Standards Wales packages also contain resources derived from HL7 FHIR UK Core where these exist. 

To enable the use of data elements from HL7 FHIR R5 into HL7 FHIR R4 in a standardised way, FHIR Standards Wales packages have also introduced a dependency on an HL7 FHIR backport extension.
