### {{page-title}}

These concept map files provide mapping between HL7/UK Core Code Systems and published Data Standard Wales data standards. These files should be used as appropriate on a per implementation basis.

In general, it is expected for NHS Wales source systems to comply with the data standard which can then be mapped to the appropriate HL7/UK Core concept. The inverse is true where supplier systems have implemented HL7/UK Core concepts and the equivalent Data Standard Wales value is required. It is understood that some systems may differ to what is described and they are encouraged to reach out to the {{pagelink:Help-and-Support, text:FHIR Standards Wales email inbox}} to discuss any compatibility concerns.
