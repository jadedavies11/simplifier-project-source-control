## {page-title}

There are several namespaces used throughout this guide that are UK based which can be found on the [UK Naming Systems](https://simplifier.net/guide/uknamingsystems) page. These are not maintained by NHS Wales.

This includes a range of OIDs (Object Identifiers) defined by HL7 UK outside of the FHIR specification, which can be found on their [OID Catalogue](https://www.hl7.org.uk/standards/object-identifiers-oids/hl7-uk-issued-oids/) page.
