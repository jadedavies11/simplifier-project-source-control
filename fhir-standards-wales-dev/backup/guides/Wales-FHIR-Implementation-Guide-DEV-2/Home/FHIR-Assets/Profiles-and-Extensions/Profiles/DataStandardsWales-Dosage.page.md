## {{page-title}}
The [Dosage](https://hl7.org/fhir/dosage.html) resource is a record of a medication that is administered to a patient.

The {{page-title}} profile is derived from the [HL7 Dosage Profile](https://hl7.org/fhir/dosage.html). It defines additional rules for use within health and care organisations in Wales.

### Mandatory and Must Support Data Elements
When an element is mandatory (min=1), the data is expected to always be present. Elements marked with an <span style="background-color:red;color:white;">S</span> must be supported by both producing and receiving systems and **SHOULD** be populated if the data exists.

Each MedicationAdministration must support:

1. sequence
2. text
3. additionalInstruction
4. patientInstruction
5. timing
6. asNeeded
7. site
8. route
9. method
10. maxDosePerPeriod
